2. How to run the program?

2.1 	Please install python 3.6 (or higher) using:
		$ sudo apt-get install python3.6

2.2 	Additional python packages needed to be installed:
		pandas - $ pip install pandas
		numpy - $ pip install numpy
		networkx - $ pip install networkx

2.3 	To run the program from the top, a separate script is provided. Please run assign2.sh. To run some individual program, please run the shell scripts in the top down order as listed in section 1.1 as some programs require supporting files which will be generated by some predecessing program.
		$ ./assign2.sh

2.4 	To run .py programs, please use:
		$ python3 'filename'.py

2.6 Dependencies among the programs:

2.6.1	article-categories-generator.sh uses the output files generated by article-ids-generator and category-ids-generator.

2.6.2	edges-generator.sh uses the output generated by article-ids-generator.sh

2.6.3	graph-components-generator.sh uses the output files generated by article-ids-generator.sh and edges-generator.sh

2.6.4	finished-paths-back-noback-generator.sh uses the output files generated by article-ids-generator.sh

2.6.5	percentage-paths-back-noback-generator.sh uses the output files generated by finished-paths-back-noback-generator.sh, article-ids-generator.sh and article-categories-generator.sh

2.6.6	category-paths-generator.sh and category-subtree-paths-generator.sh use the output generated by article-categories-generator.sh, category-ids-generator.sh and edges-generator.sh

2.6.7	categoy-pairs-generator.sh uses the output generated by article-categories-generator.sh and article-ids-generator.sh

2.6.8	category-ratios-generator.sh uses the output generated by category-pairs-generator.sh, finished-paths-back-no-back-generator.sh and article-categories-generator.sh
